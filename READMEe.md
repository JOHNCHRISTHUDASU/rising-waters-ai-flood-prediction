# 🌊 Rising Waters – AI Flood Prediction System

Rising Waters is an AI-powered Flood Prediction System designed to predict the likelihood of floods using machine learning techniques and weather-related parameters.

## ✨ Features

- AI-powered Flood Prediction
- Prediction History
- Model Performance Dashboard
- CSV Export
- Interactive Charts
- Responsive UI
- SQLite Database
- Cloud Ready

## 🛠️ Technologies Used

- Python
- Flask
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- SQLite
- HTML
- CSS
- JavaScript
- Bootstrap

## 🎯 Objective

This project predicts flood risk using machine learning and helps users make informed decisions through an interactive web dashboard.
