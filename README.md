# 🌊 Rising Waters – AI Flood Prediction System

Rising Waters is an AI-powered Flood Prediction System designed to predict flood risk using machine learning and weather-related parameters.

## ✨ Features

- AI-powered Flood Prediction
- Prediction History
- Model Performance Dashboard
- Interactive Charts
- CSV Export
- Responsive UI
- Secure Authentication
- Modern Dashboard

## 🛠️ Technologies Used

- React
- TypeScript
- Vite
- Tailwind CSS
- Machine Learning
- SQLite / Database
- Python (AI Model)
- REST API

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPOSITORY.git
```

2. Install dependencies

```bash
npm install
```

3. Start the project

```bash
npm run dev
```

## 📊 Project Objective

This project predicts flood risk using AI and Machine Learning based on environmental and weather parameters, helping users make informed decisions through an interactive web dashboard.
